# Física Orientada a la Programación

Repositorio para practicar física mediante ejercicios de programación, desde los más básicos hasta los más complejos.

## Contenidos // metas

- **Ejercicios Básicos**: Conceptos fundamentales de cinemática y dinámica
- **Ejercicios Intermedios**: Movimiento en 2D, fuerzas y energía
- **Ejercicios Avanzados**: Sistemas complejos y simulaciones

## Requisitos

- Python 3.8+
- NumPy, Matplotlib (opcional)

## Cómo usar

Cada carpeta contiene ejercicios con soluciones comentadas. Inicia por los básicos y avanza según tu nivel.
